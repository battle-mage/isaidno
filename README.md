# Consent Checkbox Cleaner (Google Chrome Extension)

 Untick all the ticked checkboxes.

## Features

 This extension finds all "checked" checkboxes inside a specified HTML element (cookie consent pop-up, etc)
 Allows to set custom delay between clicks.

## Usage

1. Open the cookie consent page (pop-up, etc).
2. Click the extension icon.
3. Set delay in milliseconds (default: 1).
4. Find and click on the HTML element with checkboxes.
5. Enjoy unticked checkboxes!

## Install/test locally

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select the folder with this repo.
